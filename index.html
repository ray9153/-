📖 使用说明


            

                
🎯 四种地址生成模式

                

                    1. 四级联动 + 自定义地址
                    
选择省→市→区→街道，然后输入详细地址。支持使用 * 号进行随机生成。

                    

                        示例：选择"浙江省 → 杭州市 → 拱墅区 → 长浜街道"，输入"绿城时代星火城*号楼 *栋 *单元 *室"
                    

                


                

                    2. 智能地址补全功能
                    
当您没有完整选择四级地址时，系统会基于您已选择的地址级别智能补全：

                    
    
                        
  • 只选择省份：系统会在该省份下随机选择市、区、街道
    • 
                        
  • 选择省市：系统会在该市下随机选择区、街道
    • 
                        
  • 选择省市区：系统会在该区下随机选择街道
    • 
                        
  • 完全不选择：系统会生成完全随机的地址
    • 
                    

                    

                        示例：选择"浙江省"，系统会自动在浙江省下随机选择杭州市/宁波市等，然后随机选择对应的区县和街道
                    

                


                

                    3. 自定义地址为空时随机生成
                    
选择了四级联动但没有输入详细地址时，系统会生成随机的详细地址。

                


                

                    4. 星号（*）随机生成功能
                    
在详细地址中使用 * 号，系统会智能替换为合适的随机数字：

                    
    
                        
  • *号楼 → 1-99号楼（如：25号楼）
    • 
                        
  • *栋 → 1-8栋（如：3栋）
    • 
                        
  • *单元 → 1-3单元（如：2单元）
    • 
                        
  • *号 → 100-999号（如：456号）
    • 
                        
  • *室 → 100-999室（如：789室）
    • 
                        
  • 其他* → 1-99的随机数字
    • 
                    

                

            


            

                
🔧 操作步骤

                
    
                    
  1. 设置收货地址：可选择四级联动地址或留空使用随机地址
    2. 
                    
  2. 输入详细地址：可输入具体地址或使用*号进行随机生成
    3. 
                    
  3. 设置手机号：输入4位尾号，系统会生成完整手机号
    4. 
                    
  4. 生成信息：点击"生成地址"按钮生成完整信息
    5. 
                    
  5. 复制使用：点击"一键复制"按钮复制到剪贴板
    6. 
                

            


            

                
💡 使用技巧

                
    
                    
  • 快速随机：不选择任何地址直接点击生成，获得完全随机地址
    • 
                    
  • 智能补全：只选择省份或省市，系统会智能补全下级地址，确保地理位置合理
    • 
                    
  • 半自定义：选择省市区街道，详细地址留空，获得该地区的随机详细地址
    • 
                    
  • 灵活生成：使用*号可以固定部分地址，随机生成其他部分
    • 
                    
  • 批量生成：多次点击生成按钮可获得不同的随机结果
    • 
                    
  • 地理合理性：系统确保生成的地址在地理上是真实存在的组合
    •
